# SauceDemo Playwright Automation Framework

End-to-end test automation suite for [SauceDemo](https://www.saucedemo.com),
built with Playwright and TypeScript, using the Page Object Model (POM) design pattern.

## 🛠 Tech Stack
- [Playwright](https://playwright.dev/)
- TypeScript
- Node.js

## 📁 Project Structure
SauceDemo/
├── pages/ # Page Object classes (one per app page)
│ ├── LoginPage.ts
│ ├── InventoryPage.ts
│ ├── CartPage.ts
│ ├── CheckoutPage.ts
│ ├── CheckoutOverviewPage.ts
│ └── CheckoutCompletePage.ts
├── tests/ # Test spec files
│ └── login.spec.ts
├── playwright.config.ts
├── .gitignore
└── package.json
